# LINUX INSTALLATION

Linux does not deal well with midi, so the game might not run well (if at all) out of the box. To be able to play midi on Linux (and thus play the game!), follow the instructions at https://leoriether.github.io/FPGRARS/midi/.

You may also have to run the game manually if you want to use FPGRARS's --port parameter. Just open the console and type src/fpgrars --port [portnumber] src/main.s.

If you still cannot run the game, please open an issue at https://github.com/fer-amdias/quatro/ so I can work on a fix.

# INSTALAÇÃO LINUX

O Linux não lida bem com midi, então o jogo pode não rodar bem (se rodar) no início. Para poder tocar midi no Linux (e assim jogar o jogo!), siga as instruções (em inglês) em https://leoriether.github.io/FPGRARS/midi/.

Você também pode ter que rodar o jogo manualmente se quiser usar o parâmetro --port do FPGRARS. Para isso, abra o console e digite src/fpgrars --port [numerodoport] src/main.s

Se ainda não consegue rodar o jogo, abra uma issue em https://github.com/fer-amdias/quatro/ para que eu possa consertar o problema.
